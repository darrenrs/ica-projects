CSS Zen Garden

The top web development students from across Weber School District have collaborated to create Weber Innovation Early College's very own CSS ZenGarden website. Based off the idea from csszengarden.com.

All the students will use the same HTML file but the will change the look of the website by using CSS, adding id's and classes only.
