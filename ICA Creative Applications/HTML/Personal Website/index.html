Darren's Website

Welcome to my website. This website utilizes Bootstrap to enhance CSS and jQuery to automate JS tasks. It is likely better than your website. :P

Rubric

HTML

HTML code has no known errors and is HTML5-compliant. There are 28 HTML tags. More importantly, the HTML is entirely a product of me.

  • <!DOCTYPE html> declares the document's MIME type (text/html)
  • #document root pseudo-element used by jQuery
  • <html> the entire page content
  • <head> contains document metadata
  • <body> contains visible content
  • <meta> page metadata for internal purposes
  • <title> declares page title
  • <script> links to JS scripts (2 dist, 1 personal)
  • <link> links to CSS and media
  • <video> background code animation (only in /)
  • <source> specifies the video URL (only in /)
  • <header> contains page header
  • <nav> navbar in page header
  • <footer> contains the page footer
  • <a> links
  • <img> images
  • <button> darkmode button
  • <ul> list holder
  • <li> list elements
  • <form> form for Google search
  • <input> input for Google search
  • <div> like, everything
  • <span> inline text
  • <code> displays text in a code format
  • <kbd> displays text in a code format w/ background (keyboard)
  • <p> paragraph text
  • <h1> header 1
  • <h2> header 2
  • <h3> header 3
  • <h4> header 4
  • <br> line break
  • <hr> horizontal line

CSS

Bootstrap 3.4.1 is the core library that provides most styles. I have self-coded some custom animations. The specifications behind the "animated GIF" is that it is an MP4 video file loaded onto the background of the first 100% of the screen height. Credit to StartBootstrap.com.

JavaScript

Native JavaScript DOM selectors suck (looking at you, document.getElementById()), so jQuery is imperative for these purposes. jQuery is primarily used to control dark mode and the LocalStorage key, as well as some additional mini-functions.

Overall Website Layout

The navbar is present on all pages (and even sticks to the top!) All pages conform to the same template and are stored as index.html files under their folder name, meaning no .html file extensions are seen client-side.
